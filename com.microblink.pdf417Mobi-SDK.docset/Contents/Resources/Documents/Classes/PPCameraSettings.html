autofocusInterval

Interval between forcing two camera focuses. If <= 0, forced focuses arent performed and only continuous autofocus mode will be used.

Discussion

Interval between forcing two camera focuses. If <= 0, forced focuses arent performed and only continuous autofocus mode will be used.

Default: 3.2 seconds
